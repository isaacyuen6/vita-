# Third-party notices

Phase 0 uses package-manager and container dependencies whose license metadata is retained in their distributions. Before production distribution, this file will be expanded from an automated dependency inventory and reviewed for attribution obligations.

## Exercise Dataset

Exercise names, body parts, equipment, target muscles, secondary muscles, and English instruction steps are derived from hasaneyldrm/exercises-dataset, licensed under the MIT License.

Source: https://github.com/hasaneyldrm/exercises-dataset

The dataset repository's GIF/image media is not included in this app build.
